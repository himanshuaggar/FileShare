# FILE SHARE

A file sharing application whic helps you to share file using a specially generated link and using email.

[Link to FileShare](https://fileshare-himu.netlify.app/)

## Demo

https://photos.app.goo.gl/fS2oWaxuXWYmvDKy7
## Tech Stack

**Client:** HTML , CSS , JavaScript

**Server:** Node, Express , MongoDB


## Deployment

Client Side deployed on NETLFIY

Server Side deployed on HEROKU
